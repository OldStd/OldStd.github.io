Navigation to different parts of the webpage

We can navigate to different parts of the webpage
by clicking on the 3 navigation buttons
"Home", "Products", and "About Me"
and also the button "Go To Top".

To achieve these
we need first to mark each point
to navigate to. For example, <a name="products">

Next, we need an anchor.
For example, <a href="#products">
When we click on the anchor "Products"
we go to the point marked as "products"

Go to the right pane
Try it by clicking any of the navigation buttons.

Please look for <a name=
and href= 
in the HTML codes below (from index.html):

  <div class="right-nav">
    <a class="right-nav-a" href="#home">
    <h2>Home</h2></a>
    <a class="right-nav-a" href="#products">
    <h2>Products</h2></a>
    <a class="right-nav-a" href="#resume">
    <h2>About Me</h2></a>
  </div>
