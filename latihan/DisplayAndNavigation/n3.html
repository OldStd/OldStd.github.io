Lining up the navigation buttons horizontally

We can line up the 3 nagivation buttons
"Home", "Products", and "About Me" hrorizontally
by applying the CSS "display: inline-block"

We have classified all the navigation buttons
as "right-nav-a".
We add the CSS for class "right-nav-a"
as follows: 

a.right-nav-a {
    display: inline-block;
}
