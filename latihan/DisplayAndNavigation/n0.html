Presenting problems

In the beginning, when no CSS styles are applied,
the web page looks like that as shown in the right pane,
just a few lines from top to bottom, vertically.

First, we need to show the various parts of the page
by introducing width, height, background color &/or border.

Secondly, the 3 navigation buttons, "Home", "Products", and "About Me"
are lining up on top of each other in a vertical line.
We need to line them up in a horizontal line with CSS.
