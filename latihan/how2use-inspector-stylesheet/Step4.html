1. To set the width of the container
    • Type width and press Enter
    • Type 912px and press Enter
  2. To set the height
    • Type height and press Enter
    • Type 912px and press Enter
  3. To set the border
    • Type border and press Enter
    • Type 2px solid red and press Enter
  4. To copy the css codes and transfer them to your Sublime-text files
    • hight-light the css codes in the web-page
    • copy them to the clipboard
    • paste the codes to a new file in Sublime-text.
    • and save the file as ss-container.css
    • in your folder for this application, i.e. demo in the desktop
