In the previoud screen we see the headings h1, h2, through h6 lining up vertically.

we can change them to line up horizontally by applying

display: inline-block;

and give a bit of spacing between them

by applying padding: 10px;

All these heading h1 through h6 were given a common class of hx.

If we click on the line of h1 class hx we see an item h1.hx

To get the styling for the class of hx

We simply change the item h1.hx to .hx

and then apply our styling as follows:

.hx {
    display: inline-block;
    padding: 10px;
}

We can see the desired result as shown in the right pane
