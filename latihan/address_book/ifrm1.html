Welcome!

In the middle is a simple address book application.
We can enter the name and email address of our friends
and the data are stored in our own computer
and not in some server.

When we open the Chrome browser the next time
the data can still be retrieved
by using the buttons in the right pane
to display the list entered. :-) Cool.

The JavaScripts codes used
are copied from an example for IndexedDB,
a client-side database for HTML5
